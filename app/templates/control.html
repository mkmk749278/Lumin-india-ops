Deletes signals, outcomes, suppressions and session summaries from the engine database, and resets the live engine's gate state and trade monitor. This cannot be undone — the 30-day quality window restarts from zero. Use it to wipe test/broken data before a clean measurement window.

Reset today's gates

Clears the engine's in-memory emission state for today — caps, cooldowns, direction-conflict windows, tracked signals — without deleting any stored history. Stored signals are re-read at the next engine restart, so this is a soft reset for the current session.
